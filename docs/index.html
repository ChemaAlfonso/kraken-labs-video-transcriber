How to Use

Get started in just a few simple steps

1

Download & Install

Download the latest release from GitHub and install on your Windows, macOS, or Linux system. No additional dependencies required.

2

Configure APIs

Set up your API credentials in the Configuration page. Add OpenAI API keys or configure custom OpenAI-compatible endpoints (Groq, Ollama, etc.).

3

Select Video

Choose a local video file from your computer. The app supports most video formats and will automatically extract audio for processing.

4

Configure & Process

Set your title, language, and generation prompt. Choose your preferred transcription and AI services, then click "Generate Content Index" to start processing.

5

View Results

Access your transcriptions and AI-generated content indexes from the Generations page. Results are saved locally for future reference.
